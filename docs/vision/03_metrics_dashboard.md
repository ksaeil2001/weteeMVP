# 03_지표_대시보드 (Metrics Dashboard)

## 0. 문서 메타

* 버전: v0.1 초안
* 작성일: 2025-11-25
* 작성자: 세일
* 문서 타입: 지표 정의 및 대시보드
* 목적:

  * WeTee MVP 1단계(선생님 100명, 학생 400명)를 검증하기 위한 핵심 지표를 한 곳에 정리한다.
  * 각 기능(F-001~F-008)이 실제로 어떤 문제(P1~P8)를 얼마나 줄였는지 정량적으로 추적한다.
* 범위:

  * 제품 사용 지표, 문제 해결 지표, 사용자 경험 지표, 비즈니스 지표, 시장 인사이트 지표를 포함한다.

## 1. 지표 체계 개요

이 문서에서 사용하는 지표는 다음 다섯 가지 카테고리로 나눈다.

| 카테고리    | 코드  | 설명                                  |
| ------- | --- | ----------------------------------- |
| 제품 사용   | USG | 실제로 플랫폼이 얼마나, 어떻게 사용되는지에 대한 지표      |
| 문제 해결   | PRB | P1~P8 문제들이 얼마나 완화/해결되고 있는지에 대한 지표   |
| 사용자 경험  | EXP | 사용자의 체감 경험(편의성, 시간 절감, 신뢰 등)에 대한 지표 |
| 비즈니스    | BUS | 수익성, 지속 가능성, 성장성에 대한 지표             |
| 시장 인사이트 | MKT | 시장/사용자 특성 이해를 위한 참고용 지표             |

## 4. 운영 가이드 (측정·보고 루틴)
